Range e Controle de Loops

range() e Fluxo do Loop

A função range() gera sequências numéricas para usar com for. Já break e continue permitem controlar o comportamento do loop durante a execução.

  • range(fim) Gera números de 0 até fim-1.
  • range(inicio, fim) Começa no início e vai até fim-1.
  • break Encerra o loop imediatamente.
  • continue Pula para a próxima repetição.

Dica Importante

Use break para sair de um loop quando atingir uma condição final. Use continue quando quiser ignorar apenas a iteração atual.
Exemplo em Python 
for i in range(1, 11):
        if i == 5:
                continue
        if i == 9:
                break
        print("Numero:", i)

# Saida: 1, 2, 3, 4, 6, 7, 8
Leitura do fluxo: continue ignora o 5, e break encerra o loop ao chegar no 9. Por isso o 9 não é exibido.
