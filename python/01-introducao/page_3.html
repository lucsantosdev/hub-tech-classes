Aula 01: Python

Variáveis e Tipos de Dados

O que são variáveis?

Variáveis são espaços na memória usados para guardar informações que o programa vai reutilizar depois, como nome, idade ou resultado de uma conta.

  • name = "Lucas"
  • age = 18
  • height = 1.75
  • is_student = True

Entrada e Saída de Dados

Todo programa conversa com o usuário. Em Python, usamos input() para receber dados e print() para mostrar informações na tela.

Entrada

Use input() para fazer uma pergunta e guardar a resposta digitada.

Saída

Use print() para exibir mensagens, resultados e feedback do programa.

Dica Importante

O valor vindo de input() chega como texto. Quando precisar fazer contas, converta para int() ou float().

Tipos de Dados

  • int Números inteiros, como 7, 18 e 2026.
  • float Números com casas decimais, como 7.5 ou 10.99.
  • string Textos entre aspas, como "Lucas" ou "Python".
  • boolean Valores lógicos: True ou False.
Escolher o tipo certo ajuda o programa a funcionar como esperado.
Exemplo em Python 
name = input("Qual o seu nome? ")
age = int(input("Quantos anos você tem? "))
print("Olá", name, "Você tem", age, "anos de idade!")
Leitura do fluxo: o programa pergunta, o usuário responde, Python guarda os valores e depois mostra a mensagem final.
Se eu remover o int() da idade, o valor continuará sendo número ou vira texto?
