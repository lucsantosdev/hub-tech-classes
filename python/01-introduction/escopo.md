## 🎯 Objetivos:

- Mentalidade de Desenvolvedor: Solucionador de Problemas, não apenas um "coder"
- Breve História da Linguagem Python

## 📖 Conteúdo:

- Mentalidade de Desenvolvedor: Seja um Solucionador de Problemas, e não um mero codificicador.
- Como surgiu a linguagem Python
- Instalação / IDE online
- Entrada e Saída de Dados (input/output)
    - Saída: Função `print()`
    - Entrada: `input()`
- Tipos de Dados (`int`, `float`, `string`, `boolean`)

## 💡 Exemplo:

```python
name = input("Qual o seu nome? ")
age = int(input("Quantos anos você tem? "))
print("Olá", name, "Você tem ", age, "anos de idade!")
```
