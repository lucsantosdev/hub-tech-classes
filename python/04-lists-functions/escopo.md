# 📦 Aula 4 — Listas, Funções e Organização de Código

## 🎯 Objetivos:

- Pensamento Estrutural
- Reutilização de Código

## 📖 Conteúdo:

- Listas (`[]`)
- Acessando e modificando listas
- Funções (`def`)
- Parâmetros e valores retornados (`return` values)

## 💡 Exemplo:

```
def greet(name):
    return "Hello " + name

print(greet("Lucas"))
```

## 🧪 Exercícios:

- Criar uma lista de filmes favoritos
- Encontrar o maior número em uma lista
- Função que verifica se um número é par
