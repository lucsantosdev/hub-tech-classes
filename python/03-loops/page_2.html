Aula 03: Python

While e For Loops

Loop while

O while repete um bloco de código enquanto uma condição for verdadeira. Ideal quando não sabemos exatamente quantas repetições serão necessárias.

  • Estrutura: while condicao:
  • Cuidado: atualize a variável de controle para evitar loop infinito.
  • Exemplo: contador de tentativas até o palpite correto.

Loop for

O for percorre uma sequência (lista, string ou range()) e executa o bloco para cada item.

for
Itera em colecoes
range()
Gera sequencias numericas
passo
range(inicio, fim, passo)

Exemplo Prático

  • for i in range(5): print(i)  →  imprime de 0 a 4
  • while tentativas < 3:  →  repete ate atingir o limite
  • break encerra o loop e continue pula para a próxima repetição

Desafio Rápido: Faça um contador de 1 a 10 com for e depois recrie o mesmo resultado usando while.
