# 🔁 Aula 2 — Condicionais e Tomadores de Decisão

## 🎯 Objetivos:

- Pensamento Lógico
- Introdução aos tomadores de decisão

## 📖 Conteúdo:

- `if`, `elif`, `else`
- Operadores de Comparação (`>`, `<`, `==`)
- Operadores Lógicos (`and`, `or`, `not`)

## 💡 Exemplo:

```python
age = int(input("Digite sua idade: "))

if age >= 18:
    print("Você pode dirigir!")
else:
    print("Muito jovem!")
```

## 🧪 Exercícios:

- Checar número par ou ímpar
- Checar Senha
- Sistema de notas (A, B, C…)
