Aula 01 - Primeiros Passos: Conceitos Fundamentais e Contexto Geral da Área de TI

- Computador e o Programador
    - Computador é formado por Hardware e Software (explicar diferenças).
    - Software é um conjunto de instruções para o computador.
    - Exemplo de Software: Sistemas Operacionais. É o principal software de um computador. É o que permite que o usuário seja capaz de interagir com o hardware (executando outros softwares/programas, gerenciando o uso da memória, armazenamento, processamento, sistema de arquivos, etc). Os mais comuns são: Windows, MacOS e Linux.
    - Programador é o profisisonal que desenvolve (cria/mantém) software. (O computador é burro. Nós dizemos o que o computador deve fazer).
    - O que o programador/desenvolvedor pode criar/desenvolver?
        - todo tipo de software: aplicativos web, desktop, mobile, jogos, automações, chatbots, sistemas embarcados (que controlam máquinas e dispositvos específicos).
        - E as empresas pagam muito bem por esse profissional...
    - Carreiras em TI:
        - Desenvolvimento Web (Frontend, Backend, Fullstack) / Webdesign (WordPress, atua utilizando menos código).
        - Desenvolvimento Mobile, Games…
        - Infraestrutura (Hardware, Redes, CyberSecurity, Sistemas Operacionais).
